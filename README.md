# Lista de Exercícios – Lógica de Programação em C#

Este repositório contém uma coleção de exercícios desenvolvidos em **C#** com o objetivo de praticar os principais conceitos de **lógica de programação**, incluindo:

- Entrada de dados
- Processamento de informações
- Saída de resultados
- Estruturas de decisão
- Estruturas de repetição

Os exercícios foram organizados por categorias para facilitar o estudo e a prática.

---

# 📥 Entrada, Processamento e Saída de Dados

1. Criar um programa para calcular o **volume de uma caixa retangular**.

2. Criar um programa para calcular o **volume de um cilindro**.

3. Criar um programa que calcule o **consumo de combustível por quilômetro percorrido em uma viagem**, considerando:
   - Quilometragem inicial
   - Quilometragem final
   - Quantidade de combustível consumida (litros)

4. Criar um programa para **converter temperatura de Celsius para Fahrenheit**.

5. Criar um programa para calcular o **salário total de um vendedor**, considerando:
   - Salário base
   - Total de vendas
   - Percentual de comissão informado pelo usuário.

6. Criar um programa para calcular a **média harmônica das notas de um aluno**.

7. Criar um programa para calcular a **média ponderada de duas provas** realizadas por um aluno.

8. Criar um programa para **verificar se um número é primo**.

9. Criar um programa que calcule a **área de um terreno retangular**, lendo suas dimensões.

10. A padaria **Hotpão** vende pães franceses e broas diariamente.

Valores:
- Pão francês: **R$ 0,12**
- Broa: **R$ 1,50**

O programa deve:
- Ler as quantidades vendidas
- Calcular o total arrecadado
- Calcular **10% do valor** para guardar na poupança.

11. Criar um programa que leia **nome e idade** de uma pessoa e mostre quantos **dias de vida** ela possui.

Exemplo:

MARIA, VOCÊ JÁ VIVEU 6935 DIAS.

12. Criar um programa para:
- Ler o salário de um funcionário
- Aplicar **15% de aumento**
- Descontar **8% de impostos**

Exibir:
- Salário inicial
- Salário com aumento
- Salário final

# 🔀 Estruturas de Decisão

13. Criar um algoritmo que leia os valores **A, B e C** e informe se **A + B é menor que C**.

14. Ler **três valores inteiros diferentes** e mostrá-los em **ordem decrescente**.

15. Criar um programa que calcule o **IMC (Índice de Massa Corporal)**.

Fórmula:

IMC = peso / altura²

Classificação:

- Abaixo de 18,5 → Abaixo do peso  
- Entre 18,5 e 25 → Peso normal  
- Entre 25 e 30 → Acima do peso  
- Acima de 30 → Obeso  

16. Criar um programa que informe se um número é **par ou ímpar**.

17. Criar um algoritmo que leia dois valores **A e B**:

- Se forem **iguais → somar**
- Se forem **diferentes → multiplicar**

O resultado deve ser armazenado na variável **C**.

# 🔁 Estruturas de Repetição

18. Gerar e mostrar os **números ímpares entre 100 e 200**.

19. Calcular a **soma de todos os números ímpares que são múltiplos de 3** no intervalo de **1 a 500**.

20. Criar um programa que leia um valor **N (1 a 10)** e mostre sua **tabuada** no formato:

0 x N = 0
1 x N = N
2 x N = 2N
...
10 x N = 10N


21. Criar um programa que leia um valor **A** e calcule seu **fatorial (A!)**.

Exemplo:

5! = 5 × 4 × 3 × 2 × 1 = 120


22. Criar um programa que leia um número e mostre a **sequência de Fibonacci até esse número**.

23. Criar um programa que imprima os números **de 1 a 100** substituindo:

- Múltiplos de **3 → "Fizz"**
- Múltiplos de **5 → "Buzz"**
- Múltiplos de **3 e 5 → "FizzBuzz"**
